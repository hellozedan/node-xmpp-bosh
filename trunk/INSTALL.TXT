Installing node-xmpp-bosh

1. Install node.js http://nodejs.org/
(possibly at /opt/node-VERSION so that you can have multiple versions)
node-xmpp-bosh has been tested with node v0.4.x

2. Install npm: https://github.com/isaacs/npm
curl http://npmjs.org/install.sh | npm_install=rc sh

3. cd <your project directory>

4. svn checkout http://node-xmpp-bosh.googlecode.com/svn/trunk/
node-xmpp-bosh-read-only

5. Get the proxy:
npm install node-xmpp-bosh -g (should get all deps.)

6. If deps. installation fails, get them manually:
npm install tav -g
npm install ltx -g
npm install node-expat -g
npm install underscore -g
npm install node-uuid -g
npm install jsdom -g

7. Link the SVN version of the proxy to the main one (so that you can
pull and needn't wait for a release)
cd <your project directory>/node-xmpp-bosh-read-only
npm link

7a. Set the executable flag on run-server.js:
chmod +x run-server.js

8. Set the node executable in the path (if not already set):
export PATH=/opt/PATH-TO-NODE/bin/:$PATH
(also add to .bashrc)

9. Set the NODE_PATH variable:
export NODE_PATH=/opt/PATH-TO-NODE/lib/node_modules/:$NODE_PATH
(also add to .bashrc)

10. Test node:
$> node

11. Start it!!
bosh-proxy

12. If the above fails, try:
node `which bosh-proxy`

13. Report any bugs to dhruvbird@gmail.com
